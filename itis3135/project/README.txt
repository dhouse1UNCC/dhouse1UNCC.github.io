JSON FILES
comparison{
	"name": Product name
	"video": If you want to embed a youtube video, add the embed link here
	"link": Link to product website
	"description": Description/Thoughts on the product
	"image": File location of the image (needs to be in image folder)
	"imageurl": Source URL of the image
	"pros": Array of pros of the product
	"cons": Array of cons of the product
}

!!Newest posts go to the bottom!!
journal{
	"date":Date of Post
	"title":Title of the journal post
	"content":Content of the journal post
}

gallery{
	'url': File location (needs to be in image folder)
	'title': Title of the Image
	'description': Description of the Image
}
